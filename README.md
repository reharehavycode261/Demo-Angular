# Projet Demo Angular

Ce projet est une application Angular de démonstration destinée à illustrer les fonctionnalités de base du framework Angular.

## Prérequis

- Node.js (version 12 ou supérieure)
- Angular CLI

## Installation

1. Clonez le dépôt du projet depuis GitHub.
2. Naviguez dans le répertoire du projet.
3. Exécutez `npm install` pour installer toutes les dépendances nécessaires.

## Scripts npm disponibles

- `ng serve` : Démarre le serveur de développement.
- `ng build` : Compile l'application pour la production.
- `ng test` : Exécute les tests unitaires.
